# 07-decisions — Решения

## Ключевые решения

- **Python + aiogram** вместо TypeScript/Hono — Telegram-боту не нужен HTTP-фреймворк
- **Polling** вместо webhook — проще деплой, не нужен домен и SSL
- **MemoryStorage** вместо БД — MVP без персистентности
- **OpenRouter** вместо OpenAI напрямую — дешевле, json_object, доступен из РФ
