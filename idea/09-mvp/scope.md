# 09-mvp — Минимально жизнеспособный продукт

## Два режима генерации

### Режим 1: Базовый (без ссылки)
Селлер → описывает товар → AI из встроенных знаний анализирует нишу → готовая продающая стратегия.

**Длительность:** ~30 секунд.
**Данные:** только описание товара от селлера.

### Режим 2: С конкурентным анализом (со ссылкой)
Селлер → описывает товар + даёт ссылку на топ-конкурента → AI сравнивает → находит точки обхода → стратегия с учётом конкурента.

**Длительность:** ~45 секунд (AI извлекает данные из ссылки).
**Данные:** описание товара + URL карточки конкурента.

## Скоуп MVP

### Входит
- [x] Telegram-бот (aiogram, polling, MemoryStorage)
- [x] 7 категорий товаров
- [x] Ввод описания (до 2000 символов)
- [x] Базовый режим: генерация стратегии из знаний AI
- [x] Продвинутый режим: + анализ конкурента по ссылке
- [x] OpenRouter (deepseek-r1) + YandexGPT fallback
- [x] 5 сообщений-результатов: WB, Ozon, Видео, Характеристики+Фото, Стратегия
- [x] Кнопки: «Новая карточка», «Копировать всё», «Усилить ссылкой»
- [x] Docker-деплой

- [ ] Платный тариф / оплата
- [ ] История генераций
- [ ] База данных (PostgreSQL)
