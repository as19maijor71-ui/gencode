**Продукт:** Ai Маркетолог (Telegram-бот)

**Слоган:** Не текст. Стратегия.

**Суть:** AI-маркетолог, который анализирует нишу, находит твоё преимущество и создаёт продающую карточку для WB и Ozon. Не генератор текста. Эксперт внутри Telegram.

AI выступает как senior-маркетолог с опытом продаж на WB/Ozon. Использует встроенные знания о рынке. Селлер опционально может дать ссылку на конкурента — AI сравнит и найдёт преимущество.

**Результат:** не просто «карточка по алгоритму», а продающая стратегия с обоснованием каждого решения.

**Стек:** Python 3.11+, aiogram, Pydantic, OpenRouter (deepseek-r1) + YandexGPT fallback.
