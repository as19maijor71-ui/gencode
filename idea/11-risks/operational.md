# Операционные риски

| Риск | Уровень | Смягчение |
|---|---|---|
| Бот упал и не перезапустился | Medium | Docker restart: always |
| Ключи API утекли | High | .env в .gitignore, gitleaks pre-commit |
| Превышен лимит токенов | Low | Мониторинг расходов, логирование каждого вызова |
| Telegram API изменился | Low | aiogram обновляется активно, сообщество большое |
