# Jobs To Be Done

## Главные jobs

1. **Создать карточку с нуля** — когда появляется новый товар, селлеру нужно описание, заголовок, ключи, характеристики. Всё сразу.

2. **Исправить неранжирующуюся карточку** — карточка есть, но не в топе. Нужно понять, что не так с алгоритмами.

3. **Обойти конкурентов** — у всех +- одинаковые карточки. Нужно преимущество, которого нет у других.

4. **Подготовить видео для Ozon** — с марта 2026 видео стало фактором №1. Нужен сценарий.

## Не jobs

- Аналитика конкурентов (это MPSTATS)
- Подсчёт юнит-экономики
- Работа с отзывами
- Логистика и поставки
