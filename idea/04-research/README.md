# 04-research — Исследование

## TL;DR

Рынок инструментов для селлеров WB/Ozon насыщен аналитикой (MPSTATS, Moneyplace, Stat4Market), но **ни один не даёт алгоритмических рекомендаций** под конкретную площадку. Ниша «интеллектуальный ассистент по ранжированию» свободна. Главная дельта: знание актуальных алгоритмов + генерация стратегии, а не текста.
