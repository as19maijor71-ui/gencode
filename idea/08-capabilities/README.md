# 08-capabilities — Способности (обновлено)

## Что умеет продукт (MVP)

### Маркетинговый анализ (из памяти AI)
- Определять категорию товара и её специфику на WB/Ozon
- Находить типичные рабочие заголовки и триггеры для этой ниши
- Понимать ценовой сегмент и портрет покупателя
- Выявлять уникальное преимущество товара и строить карточку вокруг него

### Генерация продающей карточки
- Заголовок по формуле AIDA (внимание → интерес → желание → действие)
- Описание с маркетинговыми триггерами (дефицит, соцдоказательство, выгода)
- LSI-ключи без переспама (плотность 3-5%)
- Адаптация под алгоритмы: WB — CTR + dwell time, Ozon — видео + structured data

### Видео-сценарий для Ozon
- Хронометраж 15-60 секунд
- Покадровый сценарий: что показать, что сказать
- Заточен под фактор №1 ранжирования Ozon (видео)

### Конкурентный анализ (опционально)
- Приём ссылки на топового конкурента
- Сравнение: заголовки, триггеры, структура
- Вывод: за счёт чего обойти

### Рекомендации и стратегия
- Рекомендации по фото (что на обложку, что в карусель)
- Стратегия «что делать прямо сейчас» (3-5 конкретных действий)

## Что НЕ умеет (и не должен в MVP)

- Парсить WB/Ozon
- Работать с API маркетплейсов
- Интегрироваться с MPSTATS
- Генерировать изображения
- Хранить историю генераций
- Работать с отзывами
