# Технический стек

## Выбранный стек

| Технология | Версия | Назначение | Почему |
|---|---|---|---|
| Python | 3.11+ | Язык | Стабильный, async, богатая экосистема |
| aiogram | 3.x | Telegram Bot API | Лучшая асинхронная библиотека для Telegram |
| httpx | 0.27+ | HTTP-клиент | Async, поддержка HTTP/2, таймауты |
| Pydantic | 2.5+ | Модели данных, валидация JSON | Строгая типизация, валидация AI-ответов |
| pydantic-settings | 2.1+ | Конфигурация из .env | Стандарт для Settings |
| python-dotenv | 1.0+ | Загрузка .env | Чтение переменных окружения |

## AI-провайдеры

| Провайдер | Модель | Стоимость | Преимущества |
|---|---|---|---|
| OpenRouter | deepseek/deepseek-r1 | ~$0.55/1M токенов | Дешёвый, `response_format: json_object`, хорошее качество |
| YandexGPT | yandexgpt/latest | ~₽0.5/1000 токенов | Российский, нет проблем с блокировками, fallback |

## Что НЕ выбрали

| Технология | Почему нет |
|---|---|
| FastAPI / Flask | Не нужен HTTP-сервер. Бот на polling. |
| PostgreSQL / Redis | MVP без БД. MemoryStorage для FSM. |
| React / Web | Telegram-бот — нативный интерфейс, не нужен сайт |
| OpenAI (напрямую) | Дороже OpenRouter, проблемы с доступом из РФ |
| Kandinsky / Flux | Не нужны для MVP. Генерация текста, не изображений. |
