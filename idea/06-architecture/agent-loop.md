# 06-architecture/agent-loop

## Агентный цикл (концептуально)

У Ai Маркетолог нет классического agent loop (ReAct, OODA). Продукт — не автономный агент, а однопроходный генератор:

```
ВХОД → ПОСТРОЕНИЕ ПРОМТА → ВЫЗОВ AI → ПАРСИНГ → ВЫХОД
```

Цикличность только в повторном запросе при невалидном JSON (1 retry).

## Когда появится agent loop

Фаза 3: AI-мониторинг изменений алгоритмов WB/Ozon.
- Раз в неделю поиск новостей
- Анализ изменений
- Предложение обновления модулей правил
- С подтверждением оператора → автообновление
