# Интеграции

## MVP (Фаза 1)
Интеграций нет. Telegram-бот — самодостаточный продукт. Ввод: текст от селлера (+ опциональная ссылка). Вывод: 5 сообщений в Telegram.

## Фаза 2: Копирование
Кнопка «Копировать всё» — форматирует результат в plain text, готовый к вставке в кабинет WB/Ozon.

## Фаза 3: API маркетплейсов
- **WB API:** загрузка карточки напрямую в личный кабинет продавца
- **Ozon API:** загрузка описания, характеристик, медиа
- Риски: API WB/Ozon нестабильны, требуют авторизации продавца, могут меняться без предупреждения

## Фаза 4: Интеграция с аналитикой
- **MPSTATS API:** получение данных о конкурентах для углублённого анализа
- **Seller Moon:** альтернативный источник данных
- **Web-поиск:** AI самостоятельно ищет актуальные карточки конкурентов

## Что НЕ делаем
- Не внедряем OAuth WB/Ozon в MVP (сложно, требует модерации)
- Не зависим от MPSTATS как единственного источника данных
- Не парсим WB/Ozon напрямую (риск блокировки)
