# 01-idea — Суперидея

## Суперидея (v3 — финальная)

## Ai Маркетолог — Не текст. Стратегия.

Telegram-бот, в котором AI выступает как senior-маркетолог:

1. **Анализирует рынок** из встроенных знаний о WB/Ozon
2. **Создаёт продающую карточку** с обоснованием каждого решения
3. **Следит за изменениями алгоритмов** и адаптирует выдачу динамически
4. **Даёт конкретные действия** — не текст, а инструкцию к внедрению
5. **Опционально сравнивает с конкурентом** по ссылке селлера

**Это не AI-копирайтер. Это интеллектуальный ассистент по ранжированию.**

## Четыре столпа продукта

### 1. Не копирайтер — ассистент ранжирования
Конкуренты генерируют текст. Мы генерируем **стратегию попадания в топ**.
- Обоснование каждого решения: «почему этот заголовок», «какой триггер работает»
- Селлер получает не шаблон, а понимание рынка

### 2. Динамическая адаптация под алгоритмы
Алгоритмы WB и Ozon меняются без предупреждения. Правила ранжирования вынесены в отдельные модули (`wb_algorithm.py`, `ozon_algorithm.py`). Обновление правил — 5 минут, не переписывание продукта.
- AI при каждой генерации получает актуальные правила
- Версионирование правил: видно, под какую версию алгоритма сгенерирована карточка

### 3. Конкретные действия, не текст
Результат — не «описание карточки», а **пошаговая инструкция**:
- 🟣 Для WB: «Поставь эти 3 фото на обложку. Добавь инфографику с размерами. Используй этот заголовок.»
- 🔵 Для Ozon: «Сними видео по этому сценарию. Первый кадр — крупный план. В описании — structured data, без эмодзи.»
- 📋 Характеристики: заполнены все поля (пустое = потеря трафика)

### 4. Путь к интеграции (будущее)
MVP — Telegram-бот. В дорожной карте:
- Фаза 2: копирование готового текста в буфер (сейчас)
- Фаза 3: прямая выгрузка в кабинеты WB/Ozon через их API
- Фаза 4: интеграция с MPSTATS / Seller Moon
