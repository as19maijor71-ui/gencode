# 10-phases — Фазы развития (обновлено)

## Фаза 1: MVP — Telegram-бот (май 2026)
- Базовый режим: генерация стратегии из знаний AI
- Режим с конкурентным анализом: + ссылка на конкурента
- 7 категорий товаров
- OpenRouter (deepseek-r1) + YandexGPT fallback
- Выдача: WB, Ozon, Видео, Характеристики+Фото, Стратегия
- Правила алгоритмов в отдельных модулях (версионирование)
- Docker-деплой
- Бесплатный период

## Фаза 2: Монетизация (июнь 2026)
- Авторизация пользователей
- Лимиты на бесплатные генерации (5 в день)
- Платный тариф (Telegram Stars / ЮKassa)
- Копирование результата в буфер

## Фаза 3: Интеграции (июль-август 2026)
- Выгрузка в кабинеты WB/Ozon через API
- Интеграция с MPSTATS / Seller Moon
- Веб-интерфейс (опционально)
- История генераций + избранное

## Фаза 4: B2B / Агентства (сентябрь+ 2026)
- Мультиаккаунт
- White-label
- API для партнёров
- Автоматическое отслеживание изменений алгоритмов WB/Ozon
