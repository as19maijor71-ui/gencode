# 05-principles/one-person-company

**Принцип:** Весь продукт делается одним человеком.

**Почему:** MVP без команды, без подрядчиков, без разделения ответственности. Один разработчик = один стек (Python + aiogram) = один деплой (Docker).

**Как реализовано:** Минимальный стек. Без микросервисов. Без Kubernetes. Polling вместо webhook (не нужен домен + SSL).
