# 05-principles/proactive

**Принцип:** Бот ведёт селлера, а не ждёт.

**Почему:** Селлер — не SEO-специалист. Если оставить его один на один с пустым окном ввода — он уйдёт. Бот задаёт вопросы по одному: категория → описание → опционально ссылка → готово.

**Как реализовано:** FSM с тремя состояниями. Клавиатуры на каждом шаге.
