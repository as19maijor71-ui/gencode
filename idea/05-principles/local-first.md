# 05-principles/local-first

**Принцип:** Данные пользователя не хранятся на сервере без необходимости.

**Почему:** MVP — Telegram-бот без регистрации. FSM в памяти (MemoryStorage). При перезапуске бота состояние теряется. Это приемлемо для пробного периода.

**Когда пересмотреть:** Фаза 2 — регистрация пользователей и PostgreSQL.
