# 05-principles — Принципы (обновлено)

## Маркетолог, не копирайтер
AI не пишет текст. AI выступает как senior-маркетолог с опытом продаж на WB/Ozon. Каждое решение обосновано. Результат — стратегия, а не «описание».

## Динамическая адаптация под алгоритмы
Правила ранжирования WB и Ozon вынесены в отдельные модули. Изменение правил = изменение одного файла, а не переписывание продукта. AI при каждой генерации получает актуальную версию алгоритмов. Версионирование: карточка подписана «сгенерировано под алгоритм WB v2026.1».

## Конкретные действия
Результат — не текст для чтения, а инструкция к внедрению. «Сделай это → получишь рост». Селлер копирует и применяет.

## Data-less по умолчанию
AI использует встроенные знания о рынке. Никакого парсинга, баз данных, API маркетплейсов в MVP. Промт-инжиниринг заменяет интеграции.

## Proactive
Бот ведёт селлера. Минимум усилий от пользователя. Два режима: базовый и с конкурентным анализом.

## One-person-company
Весь продукт делается одним человеком. Без команды. Стек: Python + aiogram + OpenRouter. Один деплой — Docker.
