# Планы

## Формат

```markdown
# План: {{название}} ({{дата}})

## Цель
Что должно быть достигнуто

## Задачи
- [ ] Задача 1 — критерий: ...
- [ ] Задача 2 — критерий: ...

## Риски
- ...

## Результат
(заполнить после завершения)
```
