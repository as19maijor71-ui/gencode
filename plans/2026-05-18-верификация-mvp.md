## Сквозной сценарий: полный цикл

### 1. Старт → категория → описание → подтверждение
- `/start` → сообщение с 7 категориями
- Выбрать «👗 Одежда» → поля характеристик, приглашение описать товар
- `Толстовка оверсайз из футера, тёмно-синяя, размеры S-XXL, плотность 350 г/м²` → подтверждение с кнопками «✅ Да» / «🔄 Другая категория»
- «✅ Да, всё верно» → сообщение «🎯 Конкурентный анализ» + кнопка «⏭ Пропустить»

### 2. Без конкурента: генерация → 5 сообщений
- Нажать «⏭ Пропустить»
- Анимация «🔍 Анализирую товар...»
- Результат: **5 сообщений** (🟣 WB, 🔵 Ozon, 🎬 Видео, 📋 Характеристики+Фото, 🎯 Стратегия)
- Каждое сообщение отображается без ошибок Telegram
- Заголовки, описания, ключи — непустые в каждом из 5 сообщений

### 3. Кнопка «Копировать всё» (без конкурента)
- После 5 сообщений → сообщение «📋 Нажми, чтобы скопировать...» + кнопка «📋 Копировать всё»
- Нажать кнопку → кнопка исчезает → текст карточки чистым текстом (без `<b>`, `&lt;`)
- Секции присутствуют: `=== WILDBERRIES ===`, `=== OZON ===`, `=== СТРАТЕГИЯ ===`
- Секция `=== КАК ОБОЙТИ КОНКУРЕНТА ===` **отсутствует**
- Если текст разбит на чанки — каждый ≤ 4096 символов

### 4. С конкурентом: 6 сообщений
- Новый цикл `/start` → «👗 Одежда» → «Толстовка...» → «✅ Да»
- Ввести текст конкурента (≥50 символов):
  ```
  Толстовка мужская, размеры M-L, чёрная. Состав: 80% хлопок, 20% полиэстер.
  Доставка завтра. Скидка 10% при заказе от 2 штук.
  ```
- Результат: **6 сообщений**, 6-е — «🕵️ Как обойти конкурента», непустое

### 5. Копирование с конкурентом
- Нажать «📋 Копировать всё»
- Текст содержит `=== КАК ОБОЙТИ КОНКУРЕНТА ===` с анализом

## Атомарные тесты

### 6. Кнопка «Копировать» — идемпотентность
- Сгенерировать товар А, **НЕ** нажимать «Копировать»
- Сгенерировать товар Б
- Вернуться в чат к кнопке товара А → нажать
- Ожидание: alert «⚠️ Данные устарели. Сгенерируй заново.»

### 7. Кнопка «Копировать» после рестарта бота
- Сгенерировать товар, **НЕ** нажимать «Копировать»
- Перезапустить бота
- Нажать старую кнопку «Копировать» в чате
- Ожидание: «⚠️ Данные устарели. Сгенерируй заново.» (не краш)

### 8. HTML в AI-выдаче не ломает вывод
- Сгенерировать товар с описанием, провоцирующим HTML: `крем <b>для лица</b> & рук`
- Все 5/6 сообщений отображаются без ошибок Telegram (HTML-теги экранированы)
- Если AI вернул `<b>жирный</b>` — в чате видно `<b>жирный</b>` как текст, не как форматирование

### 9. FSM не застревает при ошибке
- Сгенерировать товар → после результатов нажать любую категорию → новый товар → сгенерировать
- Ожидание: 2 полных цикла подряд без ошибок, после второго цикла можно начать третий

### 10. `/start` во время генерации
- Начать генерацию (нажать «Пропустить» или ввести конкурента)
- Пока идёт анимация — отправить `/start`
- Ожидание: «⏳ Генерация уже идёт. Подожди, пожалуйста.» (не сброс FSM)

### 11. Сообщение во время генерации
- Начать генерацию → отправить любой текст
- Ожидание: «⏳ Генерация уже идёт. Подожди, пожалуйста.»

### 12. «Начать заново» на любом шаге
- На шаге ввода описания → «🔄 Начать заново» → выбор категории
- На шаге ввода конкурента → «🔄 Начать заново» → выбор категории

### 13. Двойной клик «Копировать»
- Сгенерировать товар → быстро нажать «📋 Копировать всё» **дважды**
- Ожидание: первое нажатие — кнопка исчезает, приходят чанки. Второе — alert «Данные устарели»
- UX: чанки от первого нажатия приходят корректно, второе не крашит

### 14. Параллельность генерации и анализа
- Засечь время генерации без конкурента: ~T секунд
- Засечь время генерации с конкурентом: ~T секунд (не 2T)
- Разница не более +50% — значит `gather` работает параллельно

### 15. Конкурентный анализ не блокирует CardResult
- Отключить OpenRouter (неправильный API-ключ в `.env`) — fallback на YandexGPT должен быть
- Запустить генерацию с конкурентом
- Ожидание: основная карточка приходит (5 сообщений), даже если анализ конкурента упал

### 16. Два пользователя одновременно
- Юзер А: начать генерацию товара «Кружка»
- Юзер Б (другой Telegram-аккаунт): параллельно генерация товара «Наушники»
- Ожидание: каждый получает **свой** результат, «📋 Копировать всё» копирует **свой** текст, а не чужой
