# Ретроспектива: Первая фича — базовый режим AI Маркетолог

**Дата:** 2026-05-17 | **Длительность:** ~6 часов | **План:** `plans/2026-05-17-первая-фича.md`

## Что сделали
- Написан весь код первой фичи: ядро (config, алгоритмы, категории, промты, генератор), бот (keyboards, handlers FSM, main), инфраструктура (Dockerfile, docker-compose, .dockerignore)
- 3 фазы × цикл из 5 ходов — найдено и исправлено 12+ багов
- Протестировано 10 карточек на реальном AI (deepseek-r1 + YandexGPT fallback)
- YandexGPT подключён и работает как fallback (исправлен folder ID)
- Починена анимация ожидания (8 стадий с прогресс-баром)
- Сделана защита от неверной категории: примеры + подтверждение + детектор ключевых слов
- Настроен Windows-фикс для asyncio (SelectorEventLoop)
- Инициализирован git, сделан первый коммит

## Что AI понял с первого раза
- CardResult как центральная сущность — строго по архитектуре
- FSM-логика (category_select → product_input → confirm → generating)
- HTML-экранирование всего AI-вывода
- Разбивка сообщений > 4096 символов по переводу строки
- OpenRouter → YandexGPT fallback

## Где AI упёрся
- Анимация ожидания: 4 итерации правок. Проблема — `MessageNotModified` от Telegram при совпадении текста первого кадра с изначальным. Решение: catch `TelegramBadRequest`
- YandexGPT 400 Bad Request: folder ID не совпадал с API-ключом. Решение: извлечь правильный ID из текста ошибки
- Windows + asyncio: `ClientConnectorError`. Решение: `WindowsSelectorEventLoopPolicy`
- VPN отваливался во время сессии — внешний фактор

## Что починили в методологии
- После каждой фазы — строго цикл из 5 ходов перед переходом к следующей
- Не писать код Фазы 2 пока не пройден цикл Фазы 1 (нарушено в начале сессии, исправлено по требованию)
- Commit message на русском — ок для внутреннего проекта

## Что в долгую память
- `deepseek-r1` думает 90-180 секунд — таймаут должен быть ≥120, а не 60
- `MessageNotModified` ломает анимацию если не ловить явно
- YandexGPT folder ID ≠ folder ID в консоли — брать из URL или ошибки API
- Промт с «транслитерация» даёт мусор в Ozon-ключах — заменить на «английские SEO-ключи»
- `.env` через `load_dotenv()` + pydantic-settings без `env_file` — работает и локально и в Docker
