# Стек технологий

## Выбранный стек

| Слой | Технология | Версия | Почему |
|---|---|---|---|
| Язык | Python | 3.11+ | Стабильный async, aiogram, Pydantic |
| Бот | aiogram | 3.4+ | Лучшая async-библиотека для Telegram Bot API |
| HTTP | httpx | 0.27+ | Async HTTP-клиент, таймауты, retry |
| Валидация | Pydantic | 2.5+ | Типизация AI-ответов, валидация JSON |
| Конфиг | pydantic-settings | 2.1+ | Settings из .env |
| Переменные | python-dotenv | 1.0+ | Загрузка .env |
| AI (основной) | OpenRouter | deepseek-r1 | $0.55/1M токенов, json_object, доступен из РФ |
| AI (fallback) | YandexGPT | latest | ₽0.5/1000 токенов, российский |
| Деплой | Docker + docker-compose | — | Один контейнер |

## Структура проекта (code)

```
cardgen/
├── bot/
│   ├── __init__.py
│   ├── main.py              # точка входа, polling
│   ├── handlers.py           # FSM, команды, генерация
│   └── keyboards.py          # клавиатуры
├── engine/
│   ├── __init__.py
│   ├── generator.py          # generate_card(), call_ai(), parse_response()
│   ├── wb_algorithm.py       # правила WB (версия 2026.1)
│   └── ozon_algorithm.py     # правила Ozon (версия 2026.1+)
├── templates/
│   ├── __init__.py
│   ├── categories.py         # 7 категорий, get_category_fields()
│   └── prompts.py            # build_prompt(), PROMPT_TEMPLATE
├── config.py                 # pydantic Settings
└── requirements.txt
├── Dockerfile
├── docker-compose.yml
```
