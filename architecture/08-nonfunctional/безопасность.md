# Безопасность

## Модель угроз (MVP)

| Угроза | Уровень | Защита |
|---|---|---|
| Утечка BOT_TOKEN | Critical | .env в .gitignore. gitleaks в pre-commit. |
| Утечка AI API ключей | Critical | .env, не в коде, не в логах, не в истории git |
| Инъекция в промт | Medium | Валидация длины ввода (2000 символов). Сырой ввод не оценивается как код. |
| Селлер шлёт вредоносный URL | Medium | URL передаётся AI для анализа, не выполняется на сервере. AI извлекает текст, не исполняет JS. |
| Перебор запросов (бесплатные — бесконечно) | Medium | В MVP — без ограничений (бесплатный период). В Фазе 2 — лимиты. |
| AI-ключи в истории git | High | Проверка перед первым коммитом: `git log --all -- .env` |

## Стоп-команды (на будущее)

Для OpenCode (наше IDE) стоп-команды задаются в конфигурации. При деплое на сервер — deny-список для AI-агента:

```
deny:
  - rm -rf /
  - git push --force
  - DROP DATABASE
  - DROP TABLE
```

## .env и секреты

```
BOT_TOKEN=           # обязателен
AI_PROVIDER=openrouter
OPENROUTER_API_KEY=
OPENROUTER_MODEL=deepseek/deepseek-r1
YANDEXGPT_API_KEY=
YANDEXGPT_FOLDER_ID=
MAX_INPUT_LENGTH=2000
```

- `.env` — в .gitignore, не коммитится
- `.env.example` — шаблон с пустыми значениями, коммитится
