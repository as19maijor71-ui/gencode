# Журнал архитектурных решений (ADR)

## Формат записи

```markdown
# ADR-001: Название решения

**Дата:** YYYY-MM-DD
**Статус:** предложено / принято / отклонено / заменено
**Контекст:** проблема или вопрос
**Решение:** что выбрали
**Альтернативы:** что рассматривали и почему отклонили
**Последствия:** что стало лучше, что хуже, риски
```
