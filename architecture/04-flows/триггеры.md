# Триггеры

## События, запускающие действия

| Триггер | Действие | Обработчик |
|---|---|---|
| `/start` | Приветствие, клавиатура категорий | bot/handlers.py |
| Выбор категории (callback) | Сохранить категорию, запросить описание | FSM → product_input |
| Ввод описания | Валидация, запуск генерации | FSM → generating → generate_card() |
| Генерация завершена | Форматирование 5 сообщений | handlers → вывод |
| Кнопка «Новая карточка» | Сброс FSM, возврат к выбору категории | handlers |
| Кнопка «Копировать всё» | Форматирование plain text | handlers |
| Ошибка AI (таймаут/JSON) | Повторный запрос или fallback | generator.py |

## Ночной режим
MVP — без ночного режима. Бот активен 24/7.
