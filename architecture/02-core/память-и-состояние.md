# Память и состояние

## 4 слоя памяти

### Слой 1: Сессия (MemoryStorage FSM)
- Состояние диалога: текущая категория, описание товара, URL конкурента
- Живёт пока селлер в диалоге
- При перезапуске бота — теряется

### Слой 2: Правила алгоритмов (файлы)
- `engine/wb_algorithm.py` — актуальные правила WB
- `engine/ozon_algorithm.py` — актуальные правила Ozon
- Версионированы. Живут в коде. Меняются вручную при обновлении алгоритмов.

### Слой 3: Знания AI (внутри модели)
- Встроенные знания о рынке WB/Ozon
- Не хранятся в проекте — внутри deepseek-r1 / YandexGPT
- Обновляются с каждой новой версией модели

### Слой 4: Шаблоны (файлы)
- `templates/categories.py` — 7 категорий, списки характеристик
- `templates/prompts.py` — структура промта
- Меняются редко. При добавлении новой категории.

## Что НЕ храним

- Историю генераций
- Данные пользователей
- Ключи API (в .env, не в коде)
- Промты пользователей
