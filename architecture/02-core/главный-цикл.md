# Главный цикл

## Основной цикл: от ввода до выдачи

```
1. Селлер → /start
2. Бот → «Выбери категорию» (клавиатура 7 категорий)
3. Селлер → выбирает категорию
4. Бот → «Опиши товар. Чем подробнее — тем точнее результат.»
5. Селлер → вводит описание (до 2000 символов)
6. [ОПЦИОНАЛЬНО] Бот → «Хочешь усилить анализ? Кинь ссылку на топ-конкурента.»
7. [ОПЦИОНАЛЬНО] Селлер → URL карточки WB или Ozon
8. Бот → «⏳ Анализирую нишу, строю стратегию...»
9. Ядро → generate_card():
   a. Получить список характеристик для категории
   b. Построить промт (правила WB + Ozon + описание + опционально URL конкурента)
   c. Вызвать AI (OpenRouter → YandexGPT fallback)
   d. Распарсить JSON → CardResult
10. Бот → 5 сообщений:
    Сообщение 1: 🟣 WILDBERRIES (заголовок, описание, LSI-ключи)
    Сообщение 2: 🔵 OZON (заголовок, описание, SEO-ключи)
    Сообщение 3: 🎬 СЦЕНАРИЙ ВИДЕО ДЛЯ OZON
    Сообщение 4: 📋 ХАРАКТЕРИСТИКИ + 📸 РЕКОМЕНДАЦИИ ПО ФОТО
    Сообщение 5: 🎯 ЧТО ДЕЛАТЬ ПРЯМО СЕЙЧАС
11. Кнопки: [🔄 Новая карточка] [📋 Копировать всё]
```

## Два режима

### Режим A: Базовый
Селлер не даёт ссылку. AI использует встроенные знания о рынке.

### Режим B: С конкурентным анализом
Селлер даёт URL конкурента. AI извлекает данные по ссылке, сравнивает, находит точки обхода.

## Что НЕ делаем в цикле

- Не храним историю (MemoryStorage, перезапуск = сброс)
- Не требуем авторизацию
- Не ограничиваем количество генераций (бесплатный период)
