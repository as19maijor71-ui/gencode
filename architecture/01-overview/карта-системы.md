# Карта системы

```mermaid
graph TD
    subgraph "Внешний мир"
        USER[Селлер<br/>Telegram]
        COMPETITOR[Карточка конкурента<br/>URL — опционально]
    end

    subgraph "Интерфейс"
        BOT[bot/main.py<br/>aiogram polling]
        HANDLERS[bot/handlers.py<br/>FSM + команды]
        KB[bot/keyboards.py<br/>клавиатуры]
    end

    subgraph "Ядро"
        GEN[engine/generator.py<br/>generate_card]
        PROMPT[templates/prompts.py<br/>build_prompt]
        CAT[templates/categories.py<br/>категории]
        WB[engine/wb_algorithm.py<br/>правила WB 2026]
        OZON[engine/ozon_algorithm.py<br/>правила Ozon 2026+]
    end

    subgraph "Внешние AI"
        OR[OpenRouter<br/>deepseek-r1]
        YG[YandexGPT<br/>fallback]
    end

    subgraph "Конфигурация"
        CFG[config.py<br/>pydantic-settings]
    end

    USER -->|описание товара| BOT
    USER -->|URL конкурента| BOT
    BOT --> HANDLERS
    HANDLERS --> GEN
    GEN --> PROMPT
    PROMPT --> CAT
    PROMPT --> WB
    PROMPT --> OZON
    GEN -->|HTTP POST| OR
    GEN -->|fallback| YG
    OR -->|JSON ответ| GEN
    YG -->|JSON ответ| GEN
    GEN --> HANDLERS
    HANDLERS -->|5 сообщений| USER
    CFG -.-> GEN
    CFG -.-> BOT
```

## Компоненты

| Компонент | Файл | Ответственность |
|---|---|---|
| Точка входа | `bot/main.py` | Запуск polling, инициализация бота |
| Хендлеры | `bot/handlers.py` | FSM, команды, обработка ввода, вывод результатов |
| Клавиатуры | `bot/keyboards.py` | Клавиатуры категорий, результатов, действий |
| Генератор | `engine/generator.py` | Вызов AI, парсинг JSON, сборка CardResult |
| Промты | `templates/prompts.py` | Сборка промта из правил + описания товара |
| Категории | `templates/categories.py` | 7 категорий, списки характеристик |
| Алгоритм WB | `engine/wb_algorithm.py` | Правила ранжирования WB (версия 2026.1) |
| Алгоритм Ozon | `engine/ozon_algorithm.py` | Правила ранжирования Ozon (версия 2026.1+) |
| Конфигурация | `config.py` | Загрузка .env, pydantic Settings |
