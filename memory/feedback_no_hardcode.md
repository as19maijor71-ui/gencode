# feedback_no_hardcode

**type:** feedback

**Правило:** Не хардкодить динамические данные в коде. Категории, правила алгоритмов, промты — в отдельных модулях (`templates/`, `engine/`), а не в строках внутри функций.

**Why:** Алгоритмы WB/Ozon меняются без предупреждения. Если правила вшиты в генератор, обновление требует переписывания кода и передеплоя. Модульная архитектура = замена 5 минут.

**How to apply:** Все изменяемые данные — в `templates/categories.py`, `templates/prompts.py`, `engine/wb_algorithm.py`, `engine/ozon_algorithm.py`. Генератор (`engine/generator.py`) только собирает, не содержит правил.
