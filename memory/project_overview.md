# project_overview

**type:** project

**Ai Маркетолог** — Telegram-бот. AI-маркетолог, который анализирует нишу селлера на WB/Ozon и создаёт продающую карточку с обоснованием каждого решения.

- Стек: Python 3.11+, aiogram, Pydantic, OpenRouter, Docker
- Деплой: polling, MemoryStorage, один контейнер
- MVP: базовый режим генерации (без ссылки на конкурента)
- Главная сущность: CardResult (Pydantic)
- Чертежи: idea/ и architecture/
